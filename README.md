# SkinTan-Dataset
SkinTan is a curated image dataset specifically designed to facilitate the study of tanned skin tones in computer vision. It aims to improve the accuracy and fairness of skin tone classification, dermatology AI, and virtual try-on technologies by providing high-quality, diverse examples of tanned skin under various lighting conditions.

The full dataset is available on Zenodo:

DOI: https://doi.org/10.5281/zenodo.19389155

Size: 477 MB  
Description: Dataset for tan skin tone image colourisation.
